**GEOAIRS** (Geoinformatics • Earth Observation • Artificial Intelligence • Remote Sensing) is an open‑source Python library designed to process **UAV imagery**, **satellite data**, **GIS layers**, and **geospatial AI models**. It brings together remote sensing, GIScience, photogrammetry, and deep learning into one unified toolkit for researchers, students, and practitioners.

<https://github.com/thapawan/surveying-land-water-aerial>

## 🛰️ **Remote Sensing**

*   Satellite imagery processing
*   Land / water / vegetation extraction
*   SAR + optical fusion workflows

## 🚁 **UAV (Drone) Processing**

*   Flight planning tools
*   UAV orthomosaic stitching
*   DEM / DSM generation
*   Vegetation indices
*   Thermal mapping

## 🌐 **GIS Tools**

*   Vector editing
*   Raster analysis
*   Hydrology tools
*   Terrain modeling
*   Automation & workflows

## 🤖 **AI & Machine Learning**

*   Object detection
*   Semantic segmentation
*   Geospatial deep learning
*   UAV deep‑learning analytics

***

## ✨ **Key Features**

*   🚁 UAV image processing: orthomosaics, DSM/DEM, vegetation indices
*   🛰️ Remote sensing tools: Sentinel, Landsat, MODIS, Planet
*   🌐 GIS workflows: vector operations, terrain, hydrology
*   🤖 AI & deep learning: detection, segmentation, canopy height models, water detection
*   🗂️ Automated dataset download tools
*   📦 Fully compatible with **QGIS**, **Jupyter**, and modern Python workflows
