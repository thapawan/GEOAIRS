# GEOAIRS
This repository for Geoinformatics Earth Observation Artificial Intelligence Remote Sensing
GEOAIRS is an open-source Python library for processing UAV imagery, satellite data, GIS layers, and geospatial AI models.

✨ Features

🚁 UAV image processing (orthomosaics, DSM/DEM, vegetation indices)
🛰️ Remote sensing tools (Sentinel, Landsat, MODIS, Planet)
🌐 GIS workflows (vector ops, terrain, hydrology)
🤖 AI & deep learning (object detection, segmentation, canopy height, water detection)
🗂️ Dataset download tools
📦 Works with QGIS & Jupyter
